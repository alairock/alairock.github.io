One thing that CakePHP 3 documentation has you do when you install is put the CakePHP Core inside the vendor directory of the app. If you want to share the CakePHP core between multiple apps you can follow these simple steps.

Step 1: Install the CakePHP 3 App

{% highlight bash %} cd {webroot} composer create-project -s dev cakephp/app ./ {% endhighlight %} Step 2: Move the CakePHP 3 core to somewhere external

In this example, I am going to put it inside `/CakeCore/cakephp` but you can put it wherever.

{% highlight bash %} mkdir /CakeCore mv {webroot}/vendor/cakephp/cakephp /CakeCore/cakephp {% endhighlight %} Step 3: Change the Core Definition in your App

Lastly we need to point the app to the newly cloned core.

{% highlight bash %} cd {webroot}/App/Config vim paths.php {% endhighlight %} And replace this line (approximately line 73):

{% highlight bash %} define('CAKE_CORE_INCLUDE_PATH', ROOT . '/vendor/cakephp/cakephp'); {% endhighlight %} With this line:

{% highlight bash %} define('CAKE_CORE_INCLUDE_PATH', '/CakeCore/cakephp/'); {% endhighlight %} That's it! Continue on your merry way.
